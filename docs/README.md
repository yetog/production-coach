## The Pitch

**One-liner:** Audiotool gives creators the studio. We help them understand what to do next.

**Problem:** DAWs give tools. They don't tell users what to do next.

**Solution:** In-DAW Production Coach that analyzes session → recommends next step → can apply changes via NEXUS.

---

## MVP Scope

1. User describes the vibe they want
2. Coach creates a dynamic production checklist
3. Coach reads session via NEXUS
4. Coach recommends next step with explanation
5. User can apply safe changes directly
